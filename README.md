## O que é NodeJs ?
```
Runtime Javascript
```

## Características
- Thread única.
- Orientado a Eventos.
- Suporte ao processamento multithread
- Instalando com o NVM para gerenciamento de versões.
- NPM
- npm init
- CommonJs
- Exportando e importando módulos.
- ES6
- Destructuring
